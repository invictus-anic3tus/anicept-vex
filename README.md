# Anicept Vex
My custom CoreXY 3D printer!

Aimed towards super speed, this printer should wow your senses. Hopefully. Maybe. You know what, do whatever. Read my journal :D

# CAD

You can see the entire CAD file at the Fusion 360 online viewer. Share link: [https://a360.co/429t4vc](https://a360.co/429t4vc)
Download the .step models in the CAD folder of this repository.

# Custom PCBs

This project uses two custom PCBs: a raspberry pi hat and a relay for 24V devices. The rpi hat fits right on top and controls a rpi cooling fan, an adxl345, and two servos.
